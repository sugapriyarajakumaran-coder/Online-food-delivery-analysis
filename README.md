# Food Delivery Data Analysis

## 📌 Project Overview
This project analyzes a multi-city food delivery platform to uncover key business insights related to revenue, customer behavior, cancellations, delivery performance, and payment trends.

## 🔍 Key Insights
- Indian cuisine is the primary revenue driver
- Hyderabad is the revenue powerhouse
- Discounts do not significantly reduce profit margin
- Late delivery is the top cancellation driver
- Delivery time is stable across cities and distances
- Card payment dominates over COD and Wallet

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## 📊 Business Impact
Provided data-driven recommendations to improve profitability, operational efficiency, and customer retention.
